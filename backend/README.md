# PeopleDraw Backend

Servidor WebSocket para la sincronización de Yjs.

## Ejecutar localmente

1. `cd backend`
2. `npm install`
3. `npm start`

El servidor estará disponible en `ws://localhost:1234`.

## Deploy

Puedes desplegar este proyecto en Fly.io o cualquier servicio que soporte Node.js y WebSockets.
